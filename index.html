Operation: Cop! é um jogo feito na Unity 3D no estilo roguelike cujo o objetivo é passar pelo maior número possível de fases sem morrer. Em cada fase, há inimigos com comportamentos diferentes. Alguns fugirão e outros tentarão atacar o personagem principal, além disso, alguns precisarão ser capturados enquanto alguns outros somente estão ali para atrapalhar. Se o jogador sofre um ataque, ele acaba perdendo energia e o mesmo acontece quando ele se move ou ataca. O jogador deve economizar energia pensando nos seus movimentos. Para recuperar energia, ele pode beber os cafés, encontrados aleatoriamente pelas fases.

O Operation: Cop! foi um trabalho universitário desenvolvido para a disciplina de Engenharia de Software cujo objetivo era desenvolver uma extensão pro editor na engine Unity 3D e ainda usar geração procedural. Para isso ser feito, todos os mapas do jogo são gerados proceduralmente, desde a quantidade de inimigos e energia até mesmo a posição de cada um dos tiles encontrados no jogo. A extensão do editor funciona de forma a definir exatamente como cada um dos inimigos vai se comportar, já que eles possuem muitos atributos para lidarmos sempre que quisermos alterar algo diretamente no código.

O enredo por trás do Operation: Cop! é simples: Você é um policial com a missão de prender o maior número de criminosos possíveis. Alguns criminosos você deverá de fato prender, como os ladrões e alguns outros vão estar ali apenas para atrapalhar sua vida, já que normalmente nada acontece com eles, como os políticos. Sua missão é sobreviver ao caos que está acontecendo na cidade, capturar os inimigos e chegar o mais longe possível sem que sua energia (representado pelo café no jogo) acabe.
